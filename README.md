# TP Python – Analyse de données EntoNova

Ce repository contient des exercices Python utilisés dans le cadre d'un TP
d'introduction à la programmation et à l'analyse de données.

Les exercices sont conçus pour des étudiants sans prérequis en informatique.

L'ensemble des exemples utilise le cas fictif **EntoNova**, une entreprise
produisant des aliments à base de protéines d'insectes.

---

# Objectifs pédagogiques

Les TP couvrent les notions suivantes :

- bases de Python (variables, listes, dictionnaires)
- boucles et conditions
- manipulation de données
- import de données (CSV, JSON, TXT)
- analyse simple avec pandas
- visualisation de données

# Utilisation

1. Cloner le repository
```
git clone https://github.com/username/tp-python-entonova
```

2. Installer les dépendances

```
pip install pandas matplotlib numpy
```

3. Lancer Jupyter Notebook à partir d'Anaconda


# Contexte pédagogique

Ces supports sont utilisés dans le cadre d'un cours d'initiation
à  l'utilisation de Python pour des étudiants non informaticiens.
